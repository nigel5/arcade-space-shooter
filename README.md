# spaceBattleGame

Here's a topdown shooter game that I made with pygame for Python3.

### Prerequisites
- python-3.7
- pygame-1.9.4 for python-3.7

### How to run the game (Windows 10)
~~~
python .\spaceBattleGame.py
~~~
